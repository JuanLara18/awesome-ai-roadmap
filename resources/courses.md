## Learning Path Suggestions

**Path 1: "I want to build AI products"**
1. fast.ai Practical Deep Learning
2. Hugging Face NLP Course
3. Full Stack LLM Bootcamp
4. DeepLearning.AI Short Courses (agents, RAG)

**Path 2: "I want to understand the math and theory"**
1. Khan Academy (math foundations)
2. Stanford CS229
3. Stanford CS231n or CS224n
4. Deep Learning Specialization

**Path 3: "I want to work in ML engineering / MLOps"**
1. Machine Learning Specialization
2. Made With ML
3. Full Stack Deep Learning
4. MLOps Specialization
