# Recommended AI/ML Books

> From beginner-friendly introductions to advanced research references. Many are available for free online.

## Beginner

| Book | Author(s) | Free? | Topics |
|---|---|---|---|
| *Python Data Science Handbook* | Jake VanderPlas | [Yes](https://jakevdp.github.io/PythonDataScienceHandbook/) | NumPy, Pandas, Matplotlib, Scikit-learn |
| *Think Stats* | Allen Downey | [Yes](https://greenteapress.com/thinkstats2/) | Statistics with Python |
| *An Introduction to Statistical Learning (ISLR)* | James, Witten, Hastie, Tibshirani | [Yes](https://www.statlearning.com/) | Statistical learning fundamentals |
| *The Hundred-Page Machine Learning Book* | Andriy Burkov | Partial | Concise ML overview |
| *Grokking Machine Learning* | Luis Serrano | No | Visual, intuitive ML explanations |

## Reading Tips

- **Start with one book per topic** — don't try to read everything
- **Free books with code** (like *Dive into Deep Learning*) let you learn by doing
- **ISLR before ESL** — read the intro version before the advanced one
- **Pair books with courses** — read the book while taking a related course for reinforcement
